# Ohjelmistotekniikan harjoitustyö: opiskelujen seurantasovellus  
Course Diary-sovellus pitää kirjaa opiskelijan sovelluksen lisäämistä kursseista, niiden suorituksista ja kuinka monta opintopistettä suoritetuista kursseista on saatu. Sovellus myös ilmoittaa, kuinka monta opintopistettä on suoritettu kandidaatin tutkielmaan vaadittavasta 180 opintopisteestä.
